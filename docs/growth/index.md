# 成长篇

成长篇整理课堂之外的资源：学生组织、社团、常用行政窗口、勤工助学和实习机会。目标是让大家知道遇到事情该找谁、想参与项目可以从哪里开始。

## 学生组织与社团

后续会整理学生组织、社团和校内活动入口，方便新同学了解不同组织的定位与参与方式。

## 校园群聊与信息渠道

后续会整理常用群聊、通知渠道和信息获取方式，减少错过重要信息的情况。

## 常用行政窗口

- 教务处（EGG）：学籍、转专业、成绩单和证明材料等事务。
- 书院/学生事务处：奖助学金、心理咨询、日常请假和学生支持。
- 信息办（IT）：账号密码、网络服务和服务器使用咨询。

## 勤工助学与实践机会

- 校内助管、TA（助教）等岗位的申请信息。
- 校外实习资源、项目渠道和经验分享。
