## 📖 项目简介

**SHTU Manual** 是一个由上科大学子自发创建、自发维护、自发贡献的校园手册，涵盖了从新生入学、校园基建、选课科研到衣食住行的全方位经验整理。

本站点致力于服务 **上海科技大学（ShanghaiTech）** 学子，涵盖了从初来乍到到如鱼得水的全周期指引。帮助大家快速适应校园环境，少走弯路，高效发展。

## 🛠️ 技术栈

- **前端框架**: [VitePress](https://vitepress.dev/) 
- **包管理**: `pnpm`

## 💻 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建并预览
pnpm docs:build
pnpm docs:preview
```
